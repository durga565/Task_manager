# 📌 Task Manager (C++)

## 📖 Overview
The Task Manager is a console-based application developed in C++.  
It allows users to manage tasks efficiently using a Linked List data structure and file handling for persistent storage.

The program supports adding, deleting, viewing, saving, and loading tasks.

---

## 📝 Problem Statement
Create a task manager that allows users to:
- Add tasks
- Delete tasks
- View all tasks

The system must:
- Use a **Linked List** to manage tasks dynamically.
- Store tasks in a file before exiting.
- Load tasks automatically when the program starts.

---

## 🚀 Features

### ➕ Add a Task
- Users can add a task with:
  - Title
  - Description
  - Due Date

### ❌ Delete a Task
- Users can delete a task by providing its title.

### 📋 View All Tasks
- Display all tasks along with their details.

### 📂 Load Tasks
- Load tasks from a file into the linked list when the program starts.

### 💾 Save Tasks
- Save all tasks to a file before the program exits.

---

## 🏗 Data Structures Used

### 🔗 Linked List
- Each node represents a single task.
- Allows dynamic memory allocation.
- Efficient insertion and deletion.

### 📦 Structs
- A `struct` is used to represent each task.
- Attributes:
  - Title
  - Description
  - Due Date
